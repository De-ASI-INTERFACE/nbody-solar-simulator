# 🌌 N-Body Solar System Simulator

Production-grade interactive N-body gravitational simulator with Dash UI, SciPy ODE integration, NASA JPL Horizons live ephemeris, and real-time orbital mechanics.

## Features

### v1.2.0 roadmap
- Post-Newtonian (1PN) relativistic corrections
- High-performance execution path
- Production hardening for future scaling

### v1.1.0
- Frame-by-frame animation slider
- Energy diagnostics table
- CSV export of trajectories

### v1.0.0
- Dash + Plotly 3D orbital visualization
- Live JPL Horizons ephemeris fetch
- RK45 adaptive N-body integration

## Quick Start

```bash
git clone https://github.com/De-ASI-INTERFACE/nbody-solar-simulator.git
cd nbody-solar-simulator
pip install -r requirements.txt
python app.py
```
